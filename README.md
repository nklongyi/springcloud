# springcloud
关于spring cloud基础环节的示例项目，初学者可以对照官方教程采用本案例进行学习

所有项目基于spring boot1.5.11 version，IDEA可以直接导入相关工程，注意配置文件中的配置（注册中心、消息中间件rabbitMQ、Kafka等地址和配置端口）
